# Website Promosi Produk Digital - ShowcasePro

Website single-page untuk mempromosikan produk digital "ShowcasePro" - sebuah website showcase builder.

## Fitur Website

1. **Responsif** - Tampil sempurna di semua perangkat
2. **Demo Interaktif** - Ubah warna tema secara real-time
3. **Testimonial Slider** - Rotasi testimoni otomatis
4. **Dark Mode** - Toggle mode gelap/terang
5. **Animasi Halus** - Transisi dan efek hover yang smooth

## Cara Menjalankan Lokal

1. Clone atau download folder ini
2. Buka file `index.html` di browser
3. Tidak perlu server atau instalasi khusus

## Cara Deploy ke GitHub Pages

1. Buat repository baru di GitHub dengan nama `web-promotion` (atau nama lain)
2. Upload semua file ke repository
3. Buka Settings > Pages
4. Di bagian "Source", pilih branch `main` dan folder `/ (root)`
5. Klik Save
6. Website akan live di: `https://username.github.io/web-promotion/`
